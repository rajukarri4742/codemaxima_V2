Reverse Each Value in Array


We can iterate every element in the array and then reverse the number and replace the reversed number in the same position. Consider the below for better understanding.

int a[] = {{ '{' }}12,45,67{{ '}' }}

The output will be as below
{{ '{' }}21,54,76{{ '}' }}

Step 1: we can take the first number 12 and need to reverse the 	   number and replace the number in the same place.
        so a[0] = 12
        after reverse the number.
        a[0] = 21

Step 2: we can take the next number 45 and need to reverse the   	   number and replace the number in the same place.
        so a[1] = 45
        after reverse the number.
        a[1] = 54

Step 3: we can take the next number 67 and need to reverse the  	   number and replace the number in the same place.
        so a[2] = 67
        after reverse the number.
        a[2] = 76.
