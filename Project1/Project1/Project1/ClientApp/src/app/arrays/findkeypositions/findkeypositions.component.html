Find Key Positions in Array


In this example we can find in which positions key present in array.
for example if the array size is 5 then index starts from 0 and ends with 4 based on those indexes, we can print the positions where the number 5 was present.

Eg: 	int arr[5];

	int arr[0] = 1; 
	int arr[1] = 3; 
	int arr[2] = 5; 
	int arr[3] = 5; 
	int arr[4] = 7; 

output: The number 5 is present in 2 and 3 positions in an array.
The actual output is number 5 is present in 3 and 4 positions in an array. 
But indexes are 2 and 3. So increase the index values by 1(when printing).

        first index = 2
        first index + 1 =  2 +1 = 3
        second index = 3
        second index + 1 = 3 +1 = 4

finally, the output will be 5 is present in 3 and 4 positions in an array.
