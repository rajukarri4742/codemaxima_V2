Maximum Value of an Array


In the question itself, we can understand that we need to find the maximum i.e highest number in an array.
Consider the below steps for better understanding that how we can find the maximum number.

Example: int arr[] = {{ '{' }}1,4,2,5,7{{ '}' }}

Step 1: we can take the first element in an array and assigned it to an variable max i.e max = arr[0] = 1, 
	  then we can compare 1 with remaining elements in the array.
        
	   1 > 4 (first element is comparing with second element)
        false
        
        Here, 1 is less than 4 so we will change the value in max to 4. So, max = 4.

Step 2: compare the value in max i.e 4 with the next element in an array i.e 2.

	   4 > 2 -----> true 
	
       Here, 4 is greater than 2 is correct. So no need of changing the max value because it is the highest number   	 till now.
	 Max = 4.

Step 3: compare the value in max i.e 4 with the next element in an array i.e 5.

	   4 > 5 -----> false 
	
	Here, 4 is less than 5 so we will change the value in max to 5. So, max = 5.

Step 4: compare the value in max i.e 5 with the next element in an array i.e 7.

	   5 > 7 -----> false 
	
	Here, 5 is less than 7 so we will change the value in max  	to 7. So, max = 7.		
	So, this is the end of an array and we found the number in max is 7. So print the maximum number in the array is 7.
