Find an Element in Array

We need to find if a value exists in an array or not. 
we need to compare every number in an array with the number which we want to find.
if we get matched the number in an array with our number then we can print like number found otherwise will print number not found

Example 1:

int arr[] = {{ '{' }}1,3,5,2,7{{ '}' }}

we want to find the number 5 present in an array or not then Consider the below steps.

Step 1: first we need to compare the number 5 with the first element in the array i.e 1

    ----> 5 == 1 (false) 

Step 2: then move to next number in the array i.e 3

        ----> 5 == 3 (false) 

Step 3: again move to next number i.e 5

        ----> 5 == 5 (true)
    Here, we can stop the process because we found the number 5.

Output: found the number in an array.
