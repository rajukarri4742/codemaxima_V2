Minimum Value of Array


We need to find the minimum number in an array.
Consider the below steps for better understanding that how we can find the Minimum number.

Example: int arr[] = {{ '{' }}1,4,2,5,7{{ '}' }}

Step 1: we can take the first element in an array and assigned it to an variable min i.e min = arr[0] = 1, 
	   then we can compare 1 with remaining elements in the array.
        
	   1 < 4 (first element is comparing with second element)
         true
        
        Here, 1 is less than 4 so we will change the value in min to 1. So, min = 1.

Step 2: compare the value in min i.e 1 with the next element in an array i.e 2.

	   1 < 2 -----> true 
	
	Here, 1 is less than 2 is correct. So no need of changing the min value because it is the least number till now.
	min = 1.

Step 3: compare the value in min value i.e 1 with the next element in an array i.e 5.

	   1 < 5 -----> true
	
	Here, 1 is less than 5 is correct. So no need of changing the min value because it is the least number till now.
	min = 1.

Step 4: compare the value in min i.e 1 with the next element in an array i.e 7.

	   1 < 7 -----> true
	
	Here, 1 is less than 7 so we will change the value in min to 1. So, min = 1.		
	So, this is the end of an array and we found the number in min is 1. So print the minimum number in the array is 1.
