Different Type of Character Count


Program to find the count of number of characters , number of integers and number of special characters in a String.
Characters : all alphabet letters are consider as characters.
Integers : all numeric values are consider as Integers.
Special Characters : Other than Characters and Integers remaining all are consider as Special characters.
Example :
If we have String "hello@ 123" in the given string alphabets are 'h','e','l','l','o' then total 5 characters ,numeric values are '1','2','3' then total 3 integers, Special Characters are '@',' '(space) here space is also consider as special character then total 2 special characters, then output should be 
"number of Characters 5
number of numbers 3
number of Special Characters 2"

Sample 1 :
Stirng = "CORE i5"
Output : "number of Characters 5
number of numbers 1
number of Special Characters 1"

Sample 2 :
String = "hello world"
Output "number of Characters 10
number of numbers 0
number of Special Characters 1"
