Special String 


Program to find whether the given STring is special String or not.
Special String : In a String if any two vowels are coming side by side then that String is Special String.
Example : 
If we have String "Cream" in the given String 'e' and 'a' are vowels, 'e' is present in 3rd position and 'a' is present in 4th position , so they are coming side by side , then output should be "Special String"

Sample 1 :
string = "heep"
Output : Special String

Sample 2 :
String = "stack"
Output : "Not a Special String"
