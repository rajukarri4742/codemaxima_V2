Find a Letter in String

Finding a letter in String:
Program to Find whether the given key is present in a string or not
Example :
If we have String "operations" and the key is 'r', then 'r' is present in 4th position of the "operations" then output should be "r is present in operations".

Sample 1 :
string = "hello java"
key = 'o'
output : o is present in hello java

Sample 2 :
string = "How are you"
key = 'j'
output : j is not present in How are you
