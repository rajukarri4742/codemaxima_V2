Palindrome

A palindrome number is a number that is same after reverse. 
In simple words, Palindrome is the reverse of a number is equal to its original number.

Example 1:
    If your number is 4334 then the reverse of the numver also 4334.
    The original number and it reverse values are same so it is palindrome numver.
            
Input: 
4334
8511

Output: 
4334 is a palindrome number
8511 is a not palindrome number
