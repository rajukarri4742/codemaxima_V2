Pronic Number

 
A pronic number is a number which is the product of two consecutive integers, that is, a number in the form n (n + 1).  
 For example:  
    if n=0 the n(n+1) = 0(0+1) =0  
    if n=1 the n(n+1) = 1(1+1) =2   
    if n=2 the n(n+1) = 2(2+1) =6  
    if n=3 the n(n+1) = 3(3+1) =12  
    if n=4 the n(n+1) = 4(4+1) =20
 Result is as below:
