Reverse of Number

Display the input number in reverse order as a number.
Example 1: 
         if your program reads the value 123456 from the keyboard 
         it should print to the screen the value 654321
Example 2:  
         if your program reads the value 123400 from the keyboard 
         it should print to the screen the value 4321 (NOT 004321)
              
Input:
123456
123400
1122

Output:
654321
4321
2211
